# Criar Conta

Projeto simples para criação de contas de usuário utilizando **HTML**, **CSS** e **JavaScript**.

## 📋 Funcionalidades
- Formulário de criação de conta.
- Validação básica de campos.
- Layout moderno e responsivo.

## 🚀 Tecnologias usadas
- HTML5
- CSS3
- JavaScript
